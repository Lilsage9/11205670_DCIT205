Resources page

Computer science students often need a variety of software tools to support their coursework, programming projects, and overall academic experience. Here's a list of essential software and apps for computer science students:

Code Editors

Code editors are software tools designed for writing and editing source code. They are an essential part of a developer's toolkit, providing features that enhance the coding experience and improve productivity.

Visual Studio Code (Recommended)
PyCharm
Eclipse

Version Control

Version control is a system that allows multiple people to work on a project simultaneously without interfering with each other. It helps track changes made to the code, allows for collaboration, and provides a history of the project's development.

GitHub
GitBash
